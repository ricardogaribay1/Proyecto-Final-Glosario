Attributes

Los atributos en HTML proporcionan información adicional sobre un elemento, como su comportamiento o estilo. Los atributos se colocan dentro de la etiqueta de apertura de un elemento, como class, id, href, entre otros.

Box Model

El modelo de caja (box model) en CSS describe cómo se distribuyen los elementos en una página web. Cada elemento tiene un borde, relleno (padding), contenido y margen, lo que define su tamaño y espacio alrededor de él.

Cascading Styles

El concepto de estilos en cascada se refiere a cómo los estilos en CSS se aplican a los elementos, comenzando desde las reglas más específicas hasta las más generales. Si hay conflictos, se aplica el estilo más específico o el último definido.

Classes

Las clases en HTML se usan para aplicar estilos a múltiples elementos con una misma característica. Se asignan mediante el atributo class y se utilizan en CSS para definir reglas de estilo comunes.

Comments

Los comentarios en HTML y CSS se utilizan para agregar notas explicativas dentro del código, que no se muestran en el navegador. En HTML se usan entre <!-- y -->, y en CSS entre /* y */.

CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, colores, fuentes, espaciado y otros aspectos visuales de una página web.

DevFT Tool

DevFT Tool es una herramienta de desarrollo que facilita la implementación de pruebas, depuración y control de versiones en aplicaciones web. A menudo se utiliza para la integración continua y la automatización de tareas de desarrollo.

Displays

En CSS, la propiedad display determina cómo se muestran los elementos en la página. Los valores más comunes incluyen block, inline y flex, entre otros, y afectan la disposición de los elementos dentro de su contenedor.

Flexbox

Flexbox es un modelo de diseño de CSS que permite distribuir el espacio entre los elementos de una manera más eficiente. Se utiliza para crear diseños de página más flexibles y fluidos, especialmente cuando se necesita alinear elementos en filas o columnas.

Float

La propiedad float en CSS se utiliza para colocar un elemento a la izquierda o derecha de su contenedor, permitiendo que el contenido fluya alrededor de él. Se usa comúnmente para imágenes, menús o columnas.

Forms

Los formularios en HTML permiten a los usuarios ingresar y enviar datos a un servidor. Se crean con la etiqueta <form> y contienen elementos de entrada como <input>, <textarea> y <button>.

GitHub

GitHub es una plataforma de desarrollo colaborativo basada en Git. Permite a los desarrolladores alojar proyectos de software, colaborar en ellos y realizar el seguimiento de los cambios mediante control de versiones.

GitHub Pages

GitHub Pages es una característica de GitHub que permite alojar sitios web directamente desde un repositorio de GitHub. Es ideal para proyectos personales, documentación o páginas estáticas.

HTML

HTML (Hypertext Markup Language) es el lenguaje estándar para crear páginas web. Utiliza etiquetas para estructurar el contenido, como encabezados, párrafos, imágenes, enlaces y tablas.

ID

El atributo id en HTML se utiliza para asignar un identificador único a un elemento dentro de una página. Este identificador puede usarse para aplicar estilos específicos con CSS o para hacer referencia al elemento en JavaScript.

Inline Styles

Los estilos en línea se definen directamente en la etiqueta HTML mediante el atributo style. Esta técnica permite aplicar estilos a un solo elemento de manera rápida, pero no es recomendada para grandes proyectos debido a la falta de separación entre contenido y estilo.

Inputs

Los campos de entrada (<input>) en HTML permiten que los usuarios envíen datos a través de formularios. Existen diferentes tipos de campos de entrada, como texto, contraseñas, casillas de verificación, radio, y botones, que se definen mediante el atributo type.

Lists

Las listas en HTML se crean utilizando las etiquetas <ul> para listas no ordenadas y <ol> para listas ordenadas. Los elementos de la lista se definen con la etiqueta <li>. Estas estructuras ayudan a organizar información de manera clara y estructurada.

Margin

El margen en CSS se refiere al espacio exterior de un elemento, separando ese elemento de otros en la página. La propiedad margin puede establecerse en valores fijos (como píxeles o ems) o en porcentaje relativo al contenedor.

Padding

El padding es el espacio interno de un elemento, entre su contenido y el borde. Se define en CSS con la propiedad padding y puede aplicarse a los cuatro lados del elemento de forma individual o uniforme.

Positions

La propiedad position en CSS se utiliza para especificar cómo se posiciona un elemento en la página. Los valores más comunes incluyen static, relative, absolute, y fixed.

Semantic HTML

El HTML semántico utiliza etiquetas que tienen un significado claro y específico, como <article>, <header>, y <footer>, lo que mejora la accesibilidad y el SEO de la página web.

Specificity

La especificidad es un concepto en CSS que determina qué reglas de estilo se aplican a un elemento cuando hay varias reglas conflictivas. Se basa en un sistema de puntajes, donde los selectores más específicos tienen mayor prioridad.

Z-Index

La propiedad z-index en CSS controla el apilamiento de los elementos en la página. Determina el orden en que los elementos se colocan sobre otros, con valores más altos representando elementos más cercanos al frente.
